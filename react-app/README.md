# Haryana Dhaba & Bakes - Restaurant Website

A modern, premium restaurant website built with React and Vite. Featuring a beautiful UI with responsive design, interactive components, and optimized performance.

## 🌟 Features

- **Responsive Design** - Mobile-first approach with breakpoints for tablet and desktop
- **Modern UI/UX** - Premium aesthetic with smooth animations and transitions
- **Fast Performance** - Built with Vite for lightning-fast load times
- **SEO Optimized** - Semantic HTML and proper meta tags
- **Accessible** - WCAG compliant with proper ARIA labels
- **Interactive Components** - Smooth animations and hover effects
- **Menu System** - Dynamic menu categories and items
- **Reviews Section** - Customer testimonials and ratings
- **Contact Page** - Integrated Google Maps and contact information
- **Gallery** - Beautiful image gallery with overlay effects
- **Mobile CTA** - Floating action buttons for mobile devices

## 📱 Pages

- **Home** - Hero section with restaurant highlights
- **About** - Restaurant information and experience
- **Menu** - Categorized menu items with pricing
- **Gallery** - Restaurant ambiance and food photography
- **Reviews** - Customer testimonials and ratings
- **Contact** - Location, hours, and contact information

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/gokuworks3/Project-1.git

# Navigate to project directory
cd Project-1/react-app

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Customization

### Colors
Edit the CSS variables in `src/App.css`:
```css
:root {
  --primary-color: #3E2723;
  --accent-color: #E65100;
  /* ... more variables */
}
```

### Typography
Fonts are defined with CSS variables:
- `--title-font: 'Playfair Display'` - Headings
- `--body-font: 'Inter'` - Body text

### Spacing
Responsive spacing system using CSS custom properties:
- `--spacing-xs: 0.25rem`
- `--spacing-sm: 0.5rem`
- `--spacing-md: 1rem`
- `--spacing-lg: 1.5rem`
- `--spacing-xl: 2rem`

## 📐 Responsive Breakpoints

- **Mobile**: Default styles
- **Tablet (small)**: `@media (min-width: 576px)`
- **Tablet (large)**: `@media (min-width: 768px)`
- **Desktop**: `@media (min-width: 1024px)`

## 🔧 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS3 with Custom Properties
- **Router**: React Router DOM
- **Icons**: Boxicons
- **Fonts**: Google Fonts (Playfair Display, Inter)
